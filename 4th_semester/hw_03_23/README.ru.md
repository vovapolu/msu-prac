<b> [special_integers](./special_integers.cpp): </b><br>
    
Класс `SpecialInts`, представляет понятие "упорядоченного набора целых чисел", реализованный через массив с изменяемой длиной (длина заранее не известна, может изменяться и неограничена сверху). <br>
Для этого класса определена операция `,` (`operator , `), которая добавляет в набор еще одно число и возвращает ссылку на объект, для которого она была вызвана, чтобы можно было вызывать несколько таких операций подряд. <br>
Также для этого класса определена операция индексирования (`operator []`) для извлечения заданного элемента набора и метод `len()`, выдающий текущую длину набора. При обращении к элементам набора через операцию индексирования проверяется корректность заданного индекса. Если он меньше нуля, или, наоборот, слишком большой, то есть в наборе нет элемента с заданным номером, то выбрасывается исключения класса `BadRange`. <br>
Метод `get()` этого класса выдает значение неверного индекса, вызвавшее ошибку. <br>
Написана главная программа, которая демонстрирует корректность написанного класса. <br>
Вся выделенная динамическая память корректно освобождается.
